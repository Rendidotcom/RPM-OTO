# 📘 RPM-OTO Case Study

## Background Problem

Teachers often spend excessive time preparing repetitive curriculum and reporting documents manually.

Common issues include:
- Rewriting similar content repeatedly
- Manual document formatting
- Inefficient administrative workflows
- Time loss during supervision preparation
- Scattered educational files

These repetitive tasks reduce the time available for meaningful learning design.

---

# 🎯 Solution

RPM-OTO was developed as an educational workflow automation system.

The system transforms structured teacher input into:
- organized spreadsheet data
- automatically generated PDF documents
- structured Google Drive storage

This reduces administrative friction and improves reporting efficiency.

---
