# RPM-OTO — Educational Reporting Automation System

## 📌 Overview

RPM-OTO is an automation system designed to reduce administrative workload in educational reporting workflows.

The system automates document generation from structured input forms into standardized PDF reports.

---

## 🎯 Problem Solved

Teachers often spend excessive time:
- formatting reports manually
- generating repetitive documents
- organizing files
- handling administrative duplication

This system reduces repetitive operational tasks through automation.

---

## ⚙️ Core Workflow

Form Input → Spreadsheet Processing → PDF Generation → Drive Storage

---

## 🚀 Key Features

- automated report generation
- standardized formatting
- Google Drive integration
- lightweight workflow
- scalable structure
- reduced human error

---

## 🏫 Educational Context

Designed for real classroom and school administrative environments.

The workflow prioritizes:
- speed
- consistency
- operational efficiency
- teacher usability

---

## 🧩 Technologies Used

- Google Apps Script
- Google Sheets
- Google Drive API
- PDF automation workflow

---

## 📈 Impact

The system significantly reduces repetitive administrative workload and accelerates educational reporting processes.
