# ⚙️ RPM-OTO Tech Stack

## Core Technologies

- Google Apps Script
- HTML
- CSS
- JavaScript
- Google Spreadsheet API
- Google Drive API
- PDF Generation Workflow

---

## System Components

### 1. Input Layer
Web form interface for teacher data entry.

### 2. Processing Layer
Google Apps Script processes and validates educational data.

### 3. Spreadsheet Engine
Structured spreadsheet acts as dynamic database and workflow controller.

### 4. PDF Generator
Automatically converts structured educational content into printable PDF documents.

### 5. Cloud Storage
Generated files are automatically organized inside Google Drive.

---

## Deployment

Frontend deployed using Vercel.

---

## Educational Context

Designed for:
- Curriculum documentation
- Teacher reporting automation
- Educational administration efficiency
- Classroom planning workflows

---

## System Goal

Reduce repetitive administrative work so teachers can focus more on meaningful learning design.
